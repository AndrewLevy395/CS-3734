About this website

The Algol Meeting Scheduler was created as a final project for CS 3733: Software Engineering, a computer science class at Worcester Polytechnic Institute. Although we did some planning in advance, we completed the entire implementation in three weeks, from November 26 to December 14, 2018.

Joe Petitti made the front-end in Angular, a TypeScript-based web application framework developed by Google. The back-end utilizes Amazon Web Services. The web application interfaces with an API set up on AWS, which connects our front-end functions to AWS lambda functions. These lambda functions are Java programs, written primarily by Matt Hagan and Andy Levy, that act as the controllers for our application. The lambda functions are connected to Database Access Objects that use JDBC to connect to a MySQL database, set up and maintained by Justin Cheng, that stores all our data.

This project is released as free and open source software, under the terms of an MIT-style license. All of our source code can be viewed on GitHub.
